# Status Surface

Settings includes an Ecosystem Readiness status surface for:

- Deck Nexus
- BoardState
- Hub
- Profile
- Friends
- Notifications
- Backup
- App Links

Each section reports status, capability, availability, verification, current
owner, and future owner. The surface is intentionally explicit so local Deck
Nexus features remain usable while missing Hub features are shown honestly.

Allowed status language: active, local, verified local, planned, unavailable,
not connected, bridge required, awaiting Hub, requires setup.

Forbidden without real verification: Hub connected, live sync, friends online,
notifications active, cloud profile, or cloud backup active.
