# Capability Registry

The capability registry reports ecosystem readiness through explicit booleans:
supported, configured, verified, available, localOnly, planned, disabled, and
error.

Current verified local capabilities:

- Deck building
- Collection and owned-card tracking
- Scanner
- Import
- Export
- Local backup
- Local profile settings

Current planned or unavailable external capabilities:

- Hub identity
- Friends
- Remote notifications
- Cloud sync
- Direct BoardState launch without acknowledgment
- Hub return callbacks

No external capability may be marked verified until a real runtime or response
confirms it.
