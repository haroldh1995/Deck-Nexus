# Notification Adapter

Deck Nexus currently supports local in-app status messages only. The notification
adapter prepares typed future notification concepts without enabling remote Hub
notifications.

Future notification categories may include friend requests, shared decks,
BoardState validation completion, Advanced Gameplay completion, Dry Run
completion, backup completion, backup failure, and app updates.

Current production status:

- remote notifications unavailable
- no Hub notification queue
- no fabricated unread count
- no push subscription
- no notification service authority
