# Manual BoardState Import

When direct launch is unavailable:

1. Export the BoardState package from Deck Nexus.
2. Open Original BoardState when its import workflow is available.
3. Choose the BoardState deck snapshot import flow.
4. Select the exported package.
5. Confirm the snapshot ID and gameplay checksum.
6. Continue into Advanced Gameplay or Dry Run inside BoardState.

If BoardState import support is not available yet, the package remains a local
compatibility artifact. Deck Nexus keeps the handoff status unconfirmed until a
valid BoardState acknowledgment or return payload exists.
